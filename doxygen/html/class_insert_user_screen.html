void InsertUserScreen::Save ( wxCommandEvent &  event)

Faz o cadastro do usuário.

Parâmetros
eventEvento que ocorre ao clicar num wxButton.

Antes de salvar um usuário no banco, o método faz as seguintes checagens:

  • Verifica se todos os campos foram preenchidos;
  • Verifica se as senhas digitadas são iguais;
  • Verifica se o login informado já não está cadastrado.

Caso as informações fornecidas pelo usuário passe nos três testes, elas serão adicionadas ao banco. Uma entrada de Log também é adicionada.
