Descrição Detalhada

Gera comandos SQL.

Cria strings que representam comandos SQL de inserção, exclusão, edição, leitura e contagem a partir dos parâmetros definidos pelo usuário.

Exemplos:
SQLHandlerColumn.cpp, SQLHandlerSet.cpp, SQLHandlerTable.cpp, SQLHandlerWhere.cpp, SQLiteHandlerExec.cpp, SQLiteHandlerNumRows.cpp e SQLiteHandlerSelect.cpp.

std::string SQLHandler::DefineWhere ( std::string  column,
std::string  signal,
std::string  value 
)

Formata numa string os parâmetros where.

Parâmetros
columnNome da coluna no banco.
signalSinal de relação entre o column e value
valueValor atribuído a esta coluna.
Retorna
Uma string formatada da seguinte maneira: column signal 'value'.

É interessante a implementação de um método único de formatação, pois é nele que poderemos implementar diretrizes de segurança que impedem ataques do tipo SQL Injection.

std::string  value 
)

Define o parâmetro where de uma consulta SQL.

Parâmetros
columnNome da coluna no banco.
valueValor atribuído a esta coluna.
Retorna
A própria instância.

Adiciona o predicado numa lista de strings. Este predicado possui como padrão a preposição AND e o operador que relaciona os parâmetros column e value é o de igualdade.

Exemplos:
SQLHandlerSet.cpp, SQLHandlerWhere.cpp, SQLiteHandlerExec.cpp e SQLiteHandlerNumRows.cpp.
