Descrição Detalhada

Frame principal da aplicação.

É o frame acessado através da tela de login.

Construtores & Destrutores

LibraryScreen::LibraryScreen ( const wxString &  title,
wxApp *  app,
std::string  uid,
const wxPoint &  position = wxPoint(100, 100),
const wxSize &  size = wxSize(800, 600) 
)

Construtor.

Parâmetros
titleTítulo do frame.
*appPonteiro para a aplicação que executa o frame.
uidID do usuário que abriu o frame (através da tela de login)
positionPosição do frame na tela.
sizeTamaho do frame.

O construtor é reponsável por salvar na memória todas as informações do usuário que o acessou, inicializar a TaskBar, a barra de menus (MyMenu), a barra de ferramentas e a barra de status.
