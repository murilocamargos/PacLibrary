MENU

 

                  

 

A barra de menu detém três funções principais, 'File', 'Help' e 'User', ou correspondentes no idioma selecionado no momento em que se fez login. Cada uma tem seus sub-menus correspondentes. Seguem estes abaixo.

*Por ser muito intuitivo encontrar ajuda pela barra de menu, foi evitada a abertura de uma nova secção.

*"Como adicionar um novo usuário?" é uma pergunta para a qual existe uma resposta no livro "Outros" nesta mesma janela de Ajuda.

Uma observação a ser feita é que o menu 'User' somente é disponível para usuário no modo administrador. Neste livro podem-se encontrar questões relacionadas a esta barra.
